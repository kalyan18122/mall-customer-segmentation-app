# Mall Customer Segmentation App

Live App:
https://mall-customer-segmentation-app-j4kp4nluptp6jrpg6rlipg.streamlit.app/

## Project Overview
This project uses Machine Learning (K-Means Clustering) to segment mall customers based on:

- Gender
- Age
- Annual Income
- Spending Score

## Features
✅ Customer cluster prediction  
✅ Interactive Streamlit dashboard  
✅ Business insights  
✅ Data visualization  

## Tech Stack
Python, Streamlit, Scikit-learn, Pandas, Matplotlib
